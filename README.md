# ZerotoOne_Spring
팀 pm2.5  테스트 파트너 프로젝트의 Server 리포지리입니다.

가천대학교 2023년도 2학기 P-프로젝트 수업에 진행한 팀 프로젝트입니다.

## 프로젝트 소개
ChatGPT를 활용하여, 사용자에게 제공받은 자료를 토대로 시험 공부에 도움이 되는 학습 자료를 생성하는 앱 서비스

- 개발 기간(기획 포함): 2024.11 ~ 2024.12
